视频下载器 Android UI 优化方向

当前项目是 Kivy 单页工具：粘贴 Request URL、下载切片、合并、查看文件。下面三种方向都保留现有功能，只调整信息层级、触控、状态反馈和视觉系统。

A. 保守打磨

在现有单页结构上优化，成本最低，主要解决乱码、间距、按钮和日志可读性。
